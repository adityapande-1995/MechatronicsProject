# Launching stuff
* Global : cd FinalProject; export TURTLEBOT3_MODEL=waffle
* Launching world : roslaunch ttb3_custom.launch
* Workaround for rviz : rosrun tf static_transform_publisher 0 0 0 0 0 0 odom base_scan 50
* Workaround for rviz : rosrun tf static_transform_publisher 0 0 0 0 0 0 odom camera_rgb_optical_frame 50
* Rviz custom config : rviz -d pblart.rviz

* Launching teletop : roslaunch turtlebot3_teleop turtlebot3_teleop_key.launch
